# Server Administration

## Server access tools

The following tools are recommended for performoing remote administration from a Windows laptop.
#### SSH client
[mRemoteNG](https://mremoteng.org/) - [download](https://mremoteng.org/download). 

#### SFTP client
[WinSCP](https://winscp.net/eng/index.php) - [download](https://winscp.net/eng/download.php).
