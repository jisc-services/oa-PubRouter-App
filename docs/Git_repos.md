# Git Repositories Used by Router

Router makes use of code that is "mastered" on either GitHub or Jisc's (private) GitLab servers.  

## GitHub
| Git repository   | URL   | Contents   |
|----|----|----|
| Router   | https://github.com/jisc-services/oa-PubRouter-App   | (This repository.)   |
| Octopus   | https://github.com/jisc-services/oa-PubRouter-Octopus   | General function library.   
| Python-sword2   | https://github.com/jisc-services/oa-python-sword2   | SWORD2 library (Client & Server)   |
| Build tools   | https://github.com/jisc-services/oa-PubRouter-Build-Tools/  | Information on Jenkins and PyPi python library manager installation & configuration. Also NGINX configuration. |
| Public documentation | https://github.com/jisc-services/Public-Documentation   | Publicly accessible documentation on Router:<br>* API & notification structures<br>* info on Router's use of JATS<br>* XML structures used for SWORD2 submissions to Eprints & DSpace<br>* Router release history. |
