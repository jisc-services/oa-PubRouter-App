Publisher duplicate submissions in period

This report shows duplicate submissions (by DOI) from publishers where at least 1 duplicate was routed to at least 1 live repository account within the specified period.

Monthly versions of this report are generated automatically on the 1st of each month (for the preceding month) and can be downloaded here.

If only 1 date field is set below, then the other date is set automatically with a gap of 1 week.

In order to capture duplicates that may have occurred close to the specified from and to dates, the database is actually searched over a date range extending 7 days either side of the specified dates.

In the results, the metadata metrics strings comprise a bit_field value, followed by counts of attributes:
bit_field | authors | orcids | funders | funder-ids | grants | licenses | structured-affiliations | affiliation-ids.
Multiple metrics strings for a particular DOI indicates substantial metadata differences among the duplicates; whereas a single metrics string suggests minimal or no differences in metadata.
