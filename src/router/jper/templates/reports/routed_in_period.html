Unique items (by DOI) routed in period

This report shows the number of unique items (deduplicated by DOI) from publishers and harvested sources that were routed to at least one live repository account within the selected period.

Note: inaccuracies will arise where a duplicate DOI was routed within the date range, but the DOI was first seen before the from-date and last seen after the to-date. This is because only 2 dates are stored for each DOI: the first date and last date it was seen by Router, hence the last receipt masks any intermediate receipts that occurred within the specified range.

If only the from-date is set, then the to-date defaults to 1 year later.
If only the to-date is set, then the from-date defaults to 1 year before.

Data is retrieved for all dates in the specified range (including the from & to dates).
