Action buttons (hovering over a button will show the 'tool tip'):

  • Edit - Edit the record
  • Copy - Edit a copy of the record, which will be saved as a Draft record
  • Draft - Make record Draft
  • Live - Make record Live (can be undone by making Draft again)
  • Delete - Delete record (can be undone by making Draft again).

Only Live records will be displayed on the relevant web page. They are listed with a green background.

When an existing record is edited, the prior version is kept as a Superseded record (unless only the sort order has changed).

Both Superseded & Deleted records are purged from the database after 3 months. They are listed with text struck through, Deleted have a pale grey/pink background. They can be reinstated by changing back to Draft.

Records are ordered by sort value ascending, followed by creation date descending; so for records with the same (or no) sort value, newest first.
