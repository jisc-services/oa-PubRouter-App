Notification Details

This page shows details of a specific notification.

In particular, it provides metrics information (presence/absence of metadata, and counts of some metadata items) which is used by the duplicate handling functionality. Where the notification is a duplicate of one previously received, then key differences in the metrics between the current and prior notifications are highlighted.

It also displays the notification metadata in 2 formats:

  • Tabular presentation (highlighting any new data if a duplicate)
  • JSON (expandable/collabsible) tree.

NB. If a notification has a large number of authors the page will be very long and may take time to display.
