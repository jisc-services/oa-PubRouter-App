Batch process metrics

The page displays batch process metrics:

  • Date - Process start date
  • Start time - Process start time (GMT)
  • Server - Server on which process ran
  • Process - Name of process
  • Measure - What items are being counted
  • Count - Number of items processed
  • Seconds - Duration of batch process in seconds (3 decimal places)
  • Minutes - Duration in minutes (to 1 decimal place).

The page displays metrics data in descending datetime order according to the specified parameters, except for the special case where process *MAX Duration for All* is selected. In that case other parameters are ignored and the maximum duration recorded for each process is shown in descending duration order.

Date range: If none set then all records will be shown. If only the From-date is set, then Until-date is set far into the future (01-01-2050); conversely if only the Until-date is set, then From-date is set to 01-01-2020.


Process *MAX Duration for All* is a special case which shows the maximum duration for each of the batch processes (minimum count parameter ignored). **This query can take a while to complete.**
