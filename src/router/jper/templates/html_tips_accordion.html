HTML tips


  • Headings:  <h1>…</h1>, <h2>…</h2>, <h3>…</h3>, <h4>…</h4>
  • Paragraph:  <p>…</p>
  • Link:  <a target="_blank" href="https://…">…Link text…</a> (NB. target="_blank" causes link to open in a new tab)
  • Bullet list:  <ul> <li>…bullet-1…</li> <li>…bullet-2…</li> …</ul>
  • Bold:  <strong>…</strong> or <b>…</b>
  • Italic:  <i>…</i>
  • Line break:  <br>
  • Horizontal line:  <hr>

For more help see: https://www.w3schools.com/html/".
