How to move from Test to Live

{% if org_acc.has_sword_collection() %}

To move a repository (that uses SWORD) to live status you must obtain the following information from the institution:

  1. Live repository name
  2. Live repository URL
  3. Username of the account that will receive the deposits (via SWORD2)
  4. Password of above account
  5. Sword collection URL, preferably using secure https protocol. For EPrints the pattern is: https://your-repository-url.ac.uk/id/contents

After amending the connection settings above and clicking "Update all details", you may click "Go Live" below.
