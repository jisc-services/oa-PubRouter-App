Update institution identifiers

Identifiers are set by selecting from a drop-down selection box which is displayed when you start typing the distinguishing part of the institution name into the empty field. It uses a simple fuzzy matching to return a refined list as you enter more than 2 characters. A maximum of 10 matching values are returned.

For example, to get University of Exeter's identifier, simply type "exe".

To change an existing value, you must first delete it.

Note that the information in the greyed fields is copied from sections above. Any text shown to the right of the ID fields is that associated with the ID in the lookup table.
