Defaults (optional) - Embargo period, Post-embargo licence, Peer-reviewed indicator

Default embargo

The default embargo duration (in months), if you specify one below, will be applied when the default licence is applied, where:

  • No licence is specified in the article's metadata
  • The article metadata includes a publication date.

In such cases Publications Router will calculate an embargo end-date by adding the default embargo duration to the publication date. This calculated embargo end-date will be included in the article metadata sent to repositories. It will also be used to set the start date of the default post-embargo licence.

Default post-embargo licence

If present, the default licence information entered below will be applied whenever the article-level metadata you provide lacks any licensing metadata. (In other words, if an article already has a licence specified in its metadata then that is used; otherwise this default is added to the outgoing metadata sent to repositories.)

In those cases, in the outgoing article metadata, the start date of the post-embargo licence will be set to the embargo end date where that can be deduced (see below).

Peer reviewed indicator

If all articles submitted to Publications Router have been peer reviewed, then that may be indicated here. This information will be included in the article metadata sent to repositories.

No defaults

If you do not wish Router ever to apply a global default embargo period or post-embargo licence or peer reviewed indicator, please leave all of this section blank.
