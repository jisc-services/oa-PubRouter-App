Reporting

Monthly article distribution report, showing all the institutions that an article has been matched to.

Further information

A report is available as a CSV file that shows which of your deposited articles (identified by DOI) have been matched (routed) to particular institutions. The report has 4 columns:

  • Date the article was processed by Router
  • DOI
  • Number of institutions to which the article was matched
  • Names of recipient institutions.

For the report to be generated you must select one of the format options:

  1. One row per article, with the names of the matched institutions concatenated into a single field, semicolon separated
  2. As #1, but institution names concatenated with semicolon, followed by newline
  3. Separate rows for each institution that an article is matched to, with all columns populated (number of institutions set to 1 for each row)
  4. Separate rows for each institution that an article is matched to, with other columns populated only for each new DOI
    (E.g. if an article is matched to 3 institutions, this will generate 3 rows of data, but only the first row will show the Date, DOI, and Number of institutions; while for the following 2 rows only the institution name column will be populated, and the first 3 columns will be blank).

Sample layouts for each option are shown in this spreadsheet.

To automatically receive the report each month you must enter recipient email addresses (separated by space or comma or semicolon) in the box below.
