Matching parameters{% if curr_org_is_super %}{{org_acc.org_name}}{% endif %}

Router determines whether an article is matched to your account by comparing parameters of the types listed below with article metadata:

  • Institution identifiers (ROR, ISNI, GRID, Crossref, Ringgold)
  • Institution name variants
  • Domain names (associated with institution website/emails)
  • Postcodes
  • Grant numbers
  • Author ORCIDS
  • Author email addresses.

You can provide or change your account's matching parameters at any time by uploading a new set in a CSV file. The file can contain just one type of parameter (in a single column) or multiple types (in multiple columns), with each column's heading specifying the type of matching parameter it contains.

Possible Organisation IDs

The first search result is designated the primary organisation as it is (usually) the best match to the entered search term. The relationship (from the ROR database) of other listed organisations to the primary is indicated by a symbol:

- Primary organisation

- Parent of the primary organisation

- Child of the primary organisation

- Related to the primary organisation.

The primary organisation and any children or parent are shown with bold text.

Other symbols:

T - Hover over this symbol to see the organisation type

🌐 - Link to organisation website.

Identifiers already existing (included) in your matching parameters are highlighted green and are not selectable.

Organisations outside the UK are not listed, as they are not currently eligible to use Publications Router.

Select identifiers you want to add, then press Add selected identifiers button below.
